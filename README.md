# Streaming Platform React 🎬

A **Streaming platform** built with **React** and **Vite**, featuring a responsive UI and Firebase authentication.  

---

## 🚀 Features

- **Responsive Design** – Works on desktop, tablet, and mobile.  
- **Firebase Authentication** – Sign in and sign out securely.  
- **Dynamic Content** – Browse movies and TV shows with interactive cards.  
- **Vite + React** – Fast development and build process.  

---

## ⚡ Tech Stack

- **React** – Frontend library  
- **Vite** – Build tool for React  
- **Firebase** – Authentication backend  
- **CSS / Tailwind** – Styling and layouts
---

## 📦 Installation
1. Clone the repo:
```bash
git clone https://github.com/Y0ussef-elnagar/streaming-platform-react.git
cd streaming-platform-react
npm install 
npm run dev
```
2.Open your browser at http://localhost:5173

---

## 📄 Build & Deployment

To build the project for production:
```bash
npm run build
```
Deploy on GitHub Pages (after configuring vite.config.js base):
```bash
npm run deploy
```
Live demo: [Streaming Platform](https://y0ussef-elnagar.github.io/streaming-platform-react/)
